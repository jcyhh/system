# 安装 uni-subscribemsg 公共模块

## 方式1：通过 HBuilderX 安装（推荐）

1. 打开 HBuilderX
2. 点击菜单 `工具` → `插件安装`
3. 搜索 `uni-subscribemsg`
4. 点击安装

## 方式2：通过插件市场导入

1. 访问：https://ext.dcloud.net.cn/plugin?name=uni-subscribemsg
2. 点击 `导入插件`
3. 选择你的项目
4. 确认导入

## 方式3：通过 uniCloud web 控制台安装

1. 登录 https://unicloud.dcloud.net.cn/
2. 选择你的服务空间
3. 点击 `云函数` → `公共模块`
4. 点击 `新建公共模块`
5. 选择 `uni-subscribemsg`

## 安装后会添加以下文件：
- `uniCloud-aliyun/cloudfunctions/common/uni-subscribemsg/`
- `uniCloud-aliyun/cloudfunctions/common/uni-open-bridge-common/`（依赖）
