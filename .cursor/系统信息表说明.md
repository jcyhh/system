# 系统信息表功能说明

## 数据库表
已创建 `system_info` 表，用于存储系统协议信息。

### 表结构
```json
{
  "_id": "系统生成",
  "key": "唯一标识，如：user_agreement、privacy_policy",
  "title": "协议标题",
  "content": "协议内容（支持HTML富文本）",
  "is_published": true/false,
  "create_time": "创建时间（自动生成）",
  "update_time": "更新时间（自动生成）"
}
```

### 权限设置
- **读取权限**：所有用户可读
- **创建/更新/删除权限**：仅管理员

## 前端实现

### 页面说明
1. **user.vue** - 用户协议页面
   - 查询 `key='user_agreement'` 的数据
   - 使用 `rich-text` 组件渲染内容
   
2. **private.vue** - 隐私政策页面
   - 查询 `key='privacy_policy'` 的数据
   - 使用 `rich-text` 组件渲染内容

3. **apply.vue** - 申请页面
   - 点击《用户协议》跳转到 user.vue
   - 点击《隐私政策》跳转到 private.vue

## 使用说明

### 在uniCloud控制台添加数据
1. 打开 uniCloud Web控制台
2. 进入数据库管理
3. 找到 `system_info` 表
4. 点击"添加"按钮
5. 填写以下字段：
   - `key`: `user_agreement` 或 `privacy_policy`
   - `title`: 标题
   - `content`: 内容（支持HTML）
   - `is_published`: 勾选

### 纯文本示例
如果不需要HTML格式，直接输入纯文本即可：
```json
{
  "key": "user_agreement",
  "title": "用户协议",
  "content": "一、服务条款的确认和接纳\n本协议是用户与本平台之间关于使用本平台服务所订立的协议。\n\n二、用户信息\n1. 用户需提供真实、准确的个人信息\n2. 用户应妥善保管账号信息",
  "is_published": true
}
```

## 注意事项
1. `key` 字段必须唯一：
   - `user_agreement` - 用户协议
   - `privacy_policy` - 隐私政策
2. `content` 字段支持 HTML，可以包含样式、图片等
3. 只有 `is_published: true` 的数据才会被查询显示
4. `create_time` 和 `update_time` 会自动生成，无需手动填写
