# 公告内容示例

在云数据库 `notices` 表中手动添加公告时，`content` 字段支持以下格式：

## 1. 纯文本格式（支持换行）

```
欢迎使用联康物流系统！

本系统主要功能：
1. 在线排队登记
2. 实时查看排队进度
3. 装卸完成通知

如有问题请联系管理员。
```

## 2. HTML 格式（富文本）

```html
<h2>📢 系统升级通知</h2>

<p>尊敬的用户：</p>

<p>为了给您提供更好的服务，我们将于 <strong>2026年1月20日</strong> 进行系统升级。</p>

<h3>升级内容：</h3>
<ul>
  <li>优化排队算法，提升效率</li>
  <li>新增实时消息推送功能</li>
  <li>界面美化升级</li>
</ul>

<blockquote>
  升级期间系统可能短暂不可用，请您谅解。
</blockquote>

<p style="color: #29156a; font-weight: bold;">感谢您的支持！</p>
```

## 3. 混合格式（HTML + 图片）

```html
<h2>🎉 新功能上线</h2>

<p>我们很高兴地宣布，以下新功能已经上线：</p>

<img src="https://your-image-url.com/feature.png" alt="新功能" />

<h3>功能亮点：</h3>
<ol>
  <li><strong>实时通知</strong>：排队到号即时提醒</li>
  <li><strong>数据统计</strong>：可查看历史装卸记录</li>
  <li><strong>快速操作</strong>：一键完成装卸登记</li>
</ol>

<p>立即体验吧！</p>
```

## 4. 简单样式示例

```html
<h2 style="color: #29156a;">重要通知</h2>

<p>各位司机朋友：</p>

<div style="background-color: #fff3cd; padding: 15px; border-radius: 5px; margin: 10px 0;">
  <strong>⚠️ 注意：</strong>春节期间（1月28日-2月5日）仓库休息，请合理安排装卸时间。
</div>

<p>祝大家新春快乐，阖家幸福！</p>

<p style="text-align: right; color: #666;">
  <em>— 联康物流管理团队</em><br/>
  2026年1月15日
</p>
```

## 添加步骤

1. 打开 HBuilderX
2. uniCloud → 云数据库 → `notices` 表
3. 点击"添加"按钮
4. 填写字段：
   - `title`: 公告标题（必填）
   - `content`: 复制上面任一格式的内容（必填）
   - `is_published`: true
   - `create_time`: 当前时间戳（例如：1768441677000）
   - `update_time`: 当前时间戳
   - `user_id`: 管理员 ID（可选）

## 注意事项

- ✅ 支持所有标准 HTML 标签
- ✅ 支持内联样式
- ✅ 纯文本会自动保留换行
- ⚠️ 外部图片需要使用 HTTPS 链接
- ⚠️ 仅管理员可在云数据库添加/修改公告
